# Hand Detection using OpenCV and MediaPipe

This project implements a real-time hand detection system using Python, OpenCV, and MediaPipe.

The program accesses the webcam, detects hands in real-time, and extracts the 21 hand landmark coordinates representing finger and palm positions.

## Features
- Real-time hand detection
- Webcam-based tracking
- 21 hand landmark detection
- Finger and palm position extraction

## Technologies Used
- Python
- OpenCV
- MediaPipe

## Installation

Clone the repository:

git clone https://github.com/YOUR_USERNAME/hand-detection-opencv-mediapipe.git

Install dependencies:

pip install -r requirements.txt

Run the project:

python hand_detection.py

## Output
The program opens the webcam and detects hand landmarks in real time.
